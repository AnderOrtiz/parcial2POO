# Pequeña REST API de Gestión suscripciones

Aplicación construida con **Tyscript** + **MongoDB**, expuesta en el puerto **3000**.

---

## Variables de Entorno
Asegúrate de configurar las siguientes variables:

```
MONGODB=mongodb//unsername:password@localhost:27017
DB_NAME=anteikuDB
````

---


## Tecnologías Utilizadas

* Tyscript
* MongoDB
